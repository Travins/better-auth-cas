# Changelog

All notable changes to this project are documented in this file.

## [0.1.0] - 2026-03-19

### Added

- Initial `better-auth-cas` plugin implementation.
- Standard CAS sign-in endpoint and callback endpoint.
- CAS ticket validation via `serviceValidate`.
- Profile mapping with fallback fields and mapping resolution details.
- Optional persistence hook (`onProfileResolved`).
- Minimal Next.js example (`examples/simple-next`).
- Open-source governance docs (`CONTRIBUTING.md`, `SECURITY.md`).
